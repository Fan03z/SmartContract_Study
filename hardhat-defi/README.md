mainnet forking 意味着拷贝一份当前区块链的数据副本并放置在本地环境上继续测试,而不会影响真正的区块进程

例如: Georli Mainnet

好处: 快 , 结果真实且接近主网结果

坏处: 调用合约需要 API , 部分合约实现起来复杂还不如 mock 模拟
